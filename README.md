# We Love Movies
Front-end live: https://we-love-movies-front-6l6p.onrender.com/
Back-end live: https://we-love-movies-q050.onrender.com/movies


## Installation

1. Fork this repository by clicking the Fork button at the top right of the page.
2. Clone this repository.
3. `cd` into the newly created directory.
4. Run `npm install`.
5. Run `npm start`.

Running `npm start` will run both the backend and the client.

## Existing files

This repository is set up as a monorepo, meaning that the client and backend projects are in one repository. This allows you to open both projects in the same editor.

The table below describes the folders in this starter repository:

| Folder/file path | Description                                                      |
| ---------------- | ---------------------------------------------------------------- |
| `./back-end`      | The back-end project, which runs on `localhost:5000` by default.  |
| `./front-end`       | The front-end project, which runs on `localhost:3000` by default. |

### GET /movies

Making a `GET` request to `/movies` will return all movies.
