# WeLoveMovies Frontend Application

This is the Front End Application for the WeLoveMovies project. Follow the instructions below.

## Installation

1. Fork / clone this repository.
1. `cd front-end` navigate to the front-end folder 
1. Run `cp ./front-end/.env.development ./front-end/.env`.
1. You should not need to make changes to the `./front-end/.env` file unless you want to connect to a backend at a location other than `http://localhost:5000`.
1. Run `npm install` to install project dependencies.
1. Run `npm start` to start your server in development mode.
